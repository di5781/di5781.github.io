Apple Care warranty

As a key person in this project, I designed and developed this functionality in POS. The function of this program is to sell all the AppleCare warranties with the hardware provided from Apple in the campuses. It functioned in real time. When the AppleCare warranty was sold, and the customer information collected along with the Apple item will be posted to Apple's database in real time. Apple will create warranty agreement number pass back to POS. POS will print this number on the customers’ receipt. Also, it will create a non-receiving PO for this warranty and pass onto financial management system. So, Apple will get the payment. If it is an AppleCare warranty refunded, POS will take the refund from the register; and post a cancellation of the warranty with the original purchase transaction to Apple. At the same time, POS will create a Return to Vender through Accounting Receiving and Payable system to Apple. All the information will be displayed on Apple's ePortal in real time.
